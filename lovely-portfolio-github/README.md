## ğŸš€ Getting Started

### Prerequisites
- Node.js (v16 or higher)
- Yarn or npm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/datasciencejourney/lovely-portfolio.git
   cd lovely-portfolio
   ```

2. Install dependencies:
   ```bash
   yarn install
   # or
   npm install
   ```

3. Start the development server:
   ```bash
   yarn start
   # or
   npm start
   ```
